========================================================================
			   Hockey Fantasia 
========================================================================
Aplikasi Hockey Fantasia dirancang oleh Team Halcyon untuk memenuhi
nilai tugas akhir mata kuliah Pemrograman Game kelas A.

Perancangan aplikasi ini dilakukan di Visual Studio Community 2015
Library yang digunakan antara lain adalah SFML dan OpenGL versi 3

Kontrol : 
Player 1 (sisi bawah layar) :
-. A : menjalankan paddle ke kiri
-. D : menjalankan paddle ke kanan
-. S : meningkatkan kecepatan paddle

Player 2 (sisi atas layar) :
-. < : menjalankan paddle ke kiri
-. > : menjalankan paddle ke kanan
-. v : meningkatkan kecepatan paddle
